# FullstackJavaReact
### Employee management system

- ***Backend:*** Java (utilizes HTTP requests with a MVC- & RESTful API architecture using Spring Boot, tested with Postman)

- ***Database:*** MySQL

- ***FrontEnd:*** React + TypeScript + TailwindCSS
